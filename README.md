## Proyectos

### 🏗️ [homelab-infra](https://github.com/pedrochgdev/homelab-infra)
Infraestructura multi-nodo documentada con arquitectura clara, inventario de hardware, plan de IPs, runbooks operativos y roadmap de crecimiento. Incluye Proxmox, Ubuntu Server, Prometheus, Grafana, Ollama, Jellyfin, Raspberry Pi 5 y WireGuard.

### 🤖 Asistente Virtual 3D de Escritorio *(en desarrollo)*
Avatar 3D en tiempo real que escucha, habla, gesticula y reacciona según el contexto del usuario. Pipeline completo: NLP → motor de comportamiento → animación 3D. Diseñado para funcionar localmente, con optimización activa de CPU/GPU.

### 🛒 [Compu-Rangers](https://github.com/pedrochgdev/Compu-Rangers)
E-commerce de productos tecnológicos con arquitectura separada: backend en **Java + Spring Boot** y frontend en **C# + ASP.NET**, comunicados vía SOAP. Incluye integración con la pasarela de pagos **Izipay** (tokenización, checkout y confirmación de transacciones).

### 📦 [AprilERP Lite](https://github.com/pedrochgdev/AprilERP_Lite)
Sistema ERP de escritorio en **Java Swing** para gestión de inventario y ventas. Soporte multi-base de datos (MySQL, PostgreSQL, Oracle XE, Oracle Cloud), generación de facturas en PDF con iTextPDF y cifrado de contraseñas con jBCrypt.
